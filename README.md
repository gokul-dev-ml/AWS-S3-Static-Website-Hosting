# AWS S3 Static Website Hosting

## Project Overview

This project demonstrates the creation and configuration of an Amazon S3 bucket for static website hosting using Amazon Web Services (AWS). The objective of this project was to gain practical exposure to cloud storage services, object management, website hosting, and AWS infrastructure fundamentals.

In this project, website files including HTML, CSS, and JavaScript assets were uploaded to an AWS S3 bucket and managed through the AWS Management Console. The project provided hands-on experience with cloud-based object storage, bucket permissions, file hosting, and storage management.

# Key Concepts Learned

## Amazon S3

Amazon S3 (Simple Storage Service) is an object storage service used to store and retrieve data from anywhere over the internet.

## Object Storage

Files uploaded to S3 are stored as objects inside buckets, making cloud storage scalable and highly available.

## Static Website Hosting

S3 can host static websites by serving HTML, CSS, JavaScript, and media files directly from cloud storage.

## Bucket Permissions

Permissions control who can access files and whether content is publicly available.

## Storage Classes

Learned about AWS storage classes and how they optimize cost and performance.
